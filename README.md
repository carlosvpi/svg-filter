# svg-filter

Utility functions to create svg filters on web pages

## Install

```bash
$ npm install svg-filter
```

And then, in your code, import:

```javascript
import { filter } from 'svg-filter'
```

## Test

First bundle:

```bash
$ npm run bundle
```

And then run the tests:

```bash
$ npm test
```

## filter(svg)(props)

```javascript

filter(svg)
```
